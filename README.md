## 🎯 About the Project

This is my personal portfolio website created as a requirement for **CSPC 112 - Software Engineering** (1st Semester 2026-2027). The project showcases my skills, projects, and experience as a graduating Computer Science student preparing to enter the software engineering industry.

The design follows a **professional, clean aesthetic** suitable for job applications, internship opportunities, and networking with potential employers.

---

## ✨ Features

- **🚀 Hero Section** – Dynamic typed text effect introducing who I am
- **📖 About Me** – Personal background, education, and statistics
- **💡 Skills** – Categorized technical skills with icons (Languages, Frameworks, Tools)
- **📁 Projects** – Highlighted projects with tags and live demo links
- **📬 Contact Form** – Functional contact section with social media links
- **📱 Fully Responsive** – Works on all devices (desktop, tablet, mobile)
- **⚡ Smooth Animations** – Hover effects, transitions, and scroll behavior
- **🧭 Navigation** – Fixed navbar with smooth scrolling to sections
- **🎨 Modern UI** – Clean typography, shadows, and professional color palette
